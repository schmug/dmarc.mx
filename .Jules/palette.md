## 2026-05-07 - Add Domain Wizard ARIA live region\n**Learning:** When using dynamic hidden error containers (like `[data-wizard-error]`), screen readers will not announce the error text unless the container explicitly uses `role="alert"` and `aria-live="polite"` and is linked to the input via `aria-describedby`.\n**Action:** Always link form inputs to their corresponding error containers using `aria-describedby` and ensure the container is configured as a polite live region so that the screen reader announces the error without moving focus away from the input.
